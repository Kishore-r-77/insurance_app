Help Screen: P0033

The Template facilities regarding communication to the Client / Agent

• Template Name: Name of the template, to be user defined

• Is SMS Allowed: Is there any provision for SMS notification to the Client

This parameter is configured using item(s) YESNO in param P0050.

• Is Email Allowed: Is there any provision for E-mail notification to the Client

This parameter is configured using item(s) YESNO in param P0050.

• Is WhatsApp Allowed: Is there any provision for WhatsApp

This parameter is configured using item(s) YESNO in param P0050.

• Is Agent SMS Allowed: Do the servicing Agent's have SMS notification enabled

This parameter is configured using item(s) YESNO in param P0050.

• Is Agent Email Allowed: Do the servicing Agent's have E-mail notification enabled

This parameter is configured using item(s) YESNO in param P0050.

• Is Agent WhatsApp Allowed: Do the servicing Agent's have whatsApp enabled

This parameter is configured using item(s) YESNO in param P0050.

• Company Email: Company E-mail Id

• Company Phone: Contact Phone number of the company

• Department Name: Name of the issuing department

• Department Head: Name of the contact person in the department
