Help Screen: P0072

• Minimum Loan Amount: Minimum Loan amount allowed under the compoent.

• Max.Loan Percentage: The maximum allowable percentage of Loan. This is value will be applied on the Surrender value calculated .for the component

• Loan Interest Type: This field indicates whether Simple or compound interest to be applied on the loan amount

This parameter is configured using item(s) SimpleOrCompound in param P0050.

• Interest Payable Frequency: Loan billing freqency (eg M/Q/H/Y for Mly,Qly,Hly and Yly)

This parameter is configured using item(s) FREQ in param P0050.

• Rate of Interest: Loan interest applied on the loan amount. This is based on the effective date of the loan

• StampDutyRate: Stamp duty (in percentage) to be levied on the loan amount

• Loan Capitalization: This field indicates whether the loan capitalization is to be based on Policy anniverary or Loan Anniversary

This parameter is configured using item(s) LOANCAP in param P0050.

• CapitalizationFrequency: Loan billing freqency (eg M/Q/H/Y for Mly,Qly,Hly and Yly)

This parameter is configured using item(s) FREQ in param P0050.

• ToleranceAmount: This field specifies the allowable tolerance amount at the time of final closure of the loan

• Is previous loan to be closed: It indicates whether previous loans under the policy to be closed before issuing a fresh loan. If it is YES, the total o/s loan amount (captilized) and interest (capitalized) will be recovered from the new loan before granting a new Loan. At any point of time, there will be only one Loan active under the policy.

This parameter is configured using item(s) YESNO in param P0050.
