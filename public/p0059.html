Current or Future Benefit

This parameter is configured using item(s) CurrentOrFuture in param P0050.

• Sequence Nunmber: In case multiple transactions are performed on same day, sequence no set the order of priority in ascending order

This parameter is configured using item(s) AllocationCategory in param P0050.

This parameter is configured using column(s) in Table "accountcodes". The AccountCode or its corresponding description used for search/selection.
