Help Screen: Q0011

Allowed Coverages

• Coverage Short Name: Coverage Name

This parameter is configured using item(s) COVR in param P0050.

• Is it Mandatory?(Y/N): Is the Coverage mandatory

This parameter is configured using item(s) YESNO in param P0050.

• Basic or Rider(B/R): Is this coverage Basisc or a Rider

This parameter is configured using item(s) BASRID in param P0050.

• Term can Exceed Basic(Y/N): Can the Term exceed the Term of Basic

This parameter is configured using item(s) YESNO in param P0050.

• Premium Paying Term can Exceed Basic(Y/N): Can Premium Paying Term exceed the Premium Paying Term of Basic Coverage

This parameter is configured using item(s) YESNO in param P0050.

• Sum Assured can Exceed: Can the Term of Rider exceed the Term of Basic Coverage

This parameter is configured using item(s) YESNO in param P0050.
