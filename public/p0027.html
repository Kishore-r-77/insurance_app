Help Screen: P0027

The Accounting rules for all financial transaction codes need to be configured in this table; these rules are referred to at the time of processing the respective transactions. Multiple accounting entries could be defined for each transction code. The impacted e Account codes, sequence number and credit(-)/debit(+) details are need to be configured for each entry.

• Account Code: The generic Account code is defined for each entry

• Seq Number: This field denotes the sequence number assigned to an accounting entry in a transaction.

• GL Sign: This column represents whether the mentioned Account code has to be debited (+) or credited (-). Both credit and debit amounts for any financial ransaction should tally.

This parameter configured using an item in param P0051.
