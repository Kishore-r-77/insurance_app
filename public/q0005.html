Help Screen: Q0005

The broad parameters offered under the product

• Free look Days: Free Look cancellation could be effected since policy issuance till n days (both days inclusive)

• Maximum Lives: Maximum Lives to be included is restricted at 99 lives

• Minimum Lives: The Least number of Lives for risk coverage is limited to 1 life

• Min.period for Surrender (in months): Minimum Surrender Period

• Product Family: Policy accrues Surrender Value afer n months

This parameter configured using an item in param Q0004.

• Reinstatement from PTD (in months): Lapsed Policy can be reinstated within n months from the date of First Unpaid Premium

• Renewable: Renewable feature

This parameter is configured using item(s) YESNO in param P0050.

• Single life: Single Premium payment facility as per product features

This parameter is configured using item(s) YESNO in param P0050.

• Allowable Frequencies: Yearly, Half Yearly, Quarterly and Monthly premium payment amenities are offered

This parameter is configured using item(s) FREQ in param P0050.

• Contract Currency: Accepted Currencies are INR and USD

This parameter is configured using item(s) CCUR in param P0050.

Multiple items from the list could be selected for this parameter.

• Billing Currency: Billing under INR and USD Currencies are allowed

This parameter is configured using item(s) BCUR in param P0050.

Multiple items from the list could be selected for this parameter.

• Component Add At AnyTime flag: Additional riders, if eligible, could be appended during the term of the policy

This parameter is configured using item(s) YESNO in param P0050.

• FuturePremAdj: Provision to accept/reject future premiua due under the policy

This parameter is configured using item(s) YESNO in param P0050.

• Future Premium Adj (in years): Where ever Future Prem Adj, accepatance is restricted to a maximin of 7 years

• Grace period (in Days): This parameter is configured with the number of days to pay premium after the due date to avoid lapsaton of the policy ; If premium is not paid within this duration (in days), the policy will lapse and the status will be changed to LA.

• Billing Lead Days (Before Due date): This parameter is configured with number of days to generate premium payment notice prior to the premium due date.

• Lapse Interest: This parameter gives the interest percentage of the lapse

• Agency Channel: Channel of the Agency

This parameter is configured using item(s) AgencyChannel in param P0050.

Multiple items from the list could be selected for this parameter.

• Back Date Allowed: When backdate is allowed for this product, set the value as Y

This parameter is configured using item(s) YESNO in param P0050.

• No Lapse Guarantee: When No Lapse Guarantee is allowed for this product, set the value as Y

This parameter is configured using item(s) YESNO in param P0050.

• No Lapse Guarantee Months: No of Months that No Lapse Guarantee is valid
